ในการทำโปรเจคครั้งนั้ทางผู้จัดทำได้แบ่งการทำงานออกเป็น 2 แบบ
1. Face Recognition โดยก่อนจะทำการรันไฟล์นี้ ควรมีการติดตั้ง library ที่จำเป็นก่อน ได้แก่ 
    1. openCV
    2. CMake
    3. dlib
    ในการรันโปรแกรมนี้จะต้องเข้าไปที่ Folder ที่มีชื่อว่า venv โดยไฟล์ชื่อ face-recognition.py จะเป็นไฟล์ของโปรแกรม
    หากต้องการเพิ่มรูปภาพที่จะให้ระบบสามารถจดจำได้ ให้ไปใส่ภาพตรงของใบหน้าที่ Folder ชื่อว่า ImagesAttendance เพียงเท่านี้กฌสามารถรันไฟล์ให้ทำงานได้
    และสามารถเข้าไปดูการบันทึกผลชื่อของบุคคล วัน และเวลาที่ระบบตรวจเจอในครั้งแรกได้ใน Folder ที่มีชื่อว่า venv โดยไฟล์มีชื่อว่า record.csv

2. Face, Age and Gender Recognition ดยก่อนจะทำการรันไฟล์นี้ ควรมีการติดตั้ง library ที่จำเป็นก่อน ได้แก่ 
    1. openCV
    2. CMake
    3. dlib

    หากต้องการเพิ่มรูปภาพที่จะให้ระบบสามารถจดจำได้ ให้ไปใส่ภาพตรงของใบหน้าที่ Folder ชื่อว่า ImagesAttendance เพียงเท่านี้กฌสามารถรันไฟล์ให้ทำงานได้
    และสามารถเข้าไปดูการบันทึกผลชื่อของบุคคล วัน เวลา เพศ และอายุที่ระบบตรวจเจอในครั้งแรกได้ใน Folder ที่มีชื่อว่า venv โดยไฟล์มีชื่อว่า record.csv
